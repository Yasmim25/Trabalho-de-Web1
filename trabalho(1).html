ACESSIBILIDADE PARA WEB


Acessibilidade é tornar páginas da Web acessíveis para todos, expecialmente para pessoas com deficiência.

Para um site ser acessível, o mesmo deve ser bem objetivo.

Crie bons links, pois muitas vezes um link não contém informações necessarias para o leitor que está navegando no site.

Exemplos de links que são acessíveis, eles devem conter mais informações.

  • Saiba mais a respeito de Acessibilidade.
  • Leia mais informações sobre HTML.
  • Compre passagens para viagem aos Estados Unidos.

Exemplos de links que não devem ser usados.

  • Ler mais...
  • Acesse aqui.
  • Mostre mais.

É importante aplicar tabelas explicativas e objetivas ao seu site.
